Guia rapido

Comece em poucos minutos

Produto Abrir app

1. Setup

Suba tudo com Docker Compose

docker compose up --build

Quando o servidor subir, abra http://localhost:3001 para usar o chat, http://localhost:3001/produto para a pagina de produto e http://localhost:3001/guia para este guia.

2. Primeira conversa

  1. Crie uma aba em Nova aba.
  2. Escolha modelo, temperatura e contexto.
  3. Digite a mensagem ou use Iniciar ditado.
  4. Envie e acompanhe o streaming em tempo real.

3. Organize seu uso

  1. Duplique abas para preservar contextos diferentes.
  2. Use busca no historico para achar trechos por palavra, role ou periodo.
  3. Exporte a conversa em Markdown quando precisar compartilhar localmente.
  4. Anexe imagens ao usar modelos multimodais como llava.
  5. Acompanhe o badge de saude no app para monitorar status e latencia do modelo.

4. Solucoes rapidas e operacao

Sem resposta do modelo: confirme que o servico Ollama esta ativo e que o modelo existe localmente.
Pagina sem estilo: rode o build de CSS em web com npm run build:css.
Erro de conexao: valide se a aplicacao responde em http://localhost:3001/healthz.
Ditado ou leitura indisponivel: verifique suporte do navegador a Web Speech API.
Para uma visao executiva local, consulte o endpoint operacional http://localhost:3001/api/scorecard.

5. Proximo passo

Use o app como sua interface principal de IA local.

Se o objetivo e uso diario, abra o app, configure seu modelo preferido no onboarding e mantenha conversas separadas por tema. O historico permanece local e pesquisavel.
