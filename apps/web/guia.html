Guia rapido

Comece a usar o Meu Chat Local em 5 minutos

Acessar App Produto Guia Rápido

1Instalação

Pré-requisitos

Certifique-se de ter Node.js 20+ e Ollama instalados:

node --version  # Deve retornar v20+
ollama --version  # Deve estar instalado

Clone e instale

git clone https://github.com/seu-usuario/meu-chat-local.git
cd meu-chat-local
npm install

Configure as variáveis de ambiente

Crie um arquivo .env na raiz do projeto (opcional):

OLLAMA_BASE_URL=http://localhost:11434
NODE_ENV=development
PORT=3000

2Iniciando o Servidor

Inicie o servidor de API

cd apps/api
npm run dev

O servidor estará disponível em http://localhost:3000

💡 Dica: Certifique-se de que o Ollama está rodando antes de iniciar o servidor. 
ollama serve

3Acessando o Aplicativo

Abra seu navegador e acesse:

http://localhost:3000/app

A interface do chat estará pronta para usar.

4Operações Comuns

Criar uma nova conversa

Clique no botão + ou use o atalho Ctrl+N

Exportar conversa

Menu → Exportar → Escolher formato (JSON ou Markdown)

Importar conversa

Vá para "Importar" e selecione um arquivo JSON previously exported

Adicionar documentos (RAG)

Na conversa, clique em "Adicionar Documentos" e faça upload de arquivos

Mudar modelo de IA

Menu de configurações → Selecione um modelo do Ollama disponível

Gerenciar perfis de usuário

Admin Panel → Usuários → Criar/Editar/Deletar perfis

5Atalhos de Teclado

Atalho Ação
Ctrl+N Nova conversa
Ctrl+E Exportar conversa atual
Ctrl+/ Mostrar ajuda
Enter Enviar mensagem
Shift+Enter Quebra de linha na mensagem

6Solução de Problemas

❌ Erro: "Ollama não disponível"

Certifique-se de que Ollama está rodando:

ollama serve

E recarregue a página.

❌ Erro: "Modelo não encontrado"

Puxe um modelo do Ollama:

ollama pull llama2
ollama pull neural-chat
ollama pull mistral

❌ Histórico não está sendo salvo

Verifique as permissões do diretório de banco de dados. O servidor precisa de acesso de escrita.

❌ Backup falha ao restaurar

Se o backup está criptografado, certifique-se de que está usando a passphrase correta.

7Recursos Avançados

Modo Admin

Acesse o painel de admin em /api/docs para gerenciar:

  • Usuários e suas permissões
  • Backups e recuperação
  • Saúde do sistema e métricas
  • Auditoria de eventos

Backup Regular

Configure backups automáticos em Configurações → Backup:

# Exportar manualmente:
GET /api/backup/export?passphrase=sua_senha

Integração com Documentos

Use RAG (Retrieval-Augmented Generation) para adicionar contexto local a suas perguntas.
